# 宝宝喂养记录小程序 PRD

## 1. 产品概述

### 1.1 产品定位

宝宝喂养记录小程序是一款面向新生儿家庭自用的微信小程序，用于记录宝宝的喂养、大小便和黄疸变化，并通过每日汇总、达标判断和趋势图帮助宝妈宝爸快速了解宝宝近期状态。

第一版重点是记录方便、状态清晰、趋势直观，不作为医疗诊断工具。

### 1.2 产品目标

- 快速记录宝宝每次喂养、大小便和黄疸数据。
- 自动统计每日总奶量、喂养次数、尿便次数。
- 根据宝宝体重和日龄/月龄估算奶量是否达标。
- 根据新生儿黄疸数据自动生成风险标志。
- 通过趋势图直观看到宝宝喂养、排泄和黄疸变化。

### 1.3 第一版范围

第一版为微信小程序版本，先满足家庭自用。

包含：

- 单宝宝档案
- 喂养记录
- 大小便记录
- 黄疸记录
- 今日概览
- 趋势图
- 本地数据存储

暂不包含：

- 多宝宝管理
- 用户登录
- 云同步
- 多人协作
- 喂奶提醒
- 黄疸复查提醒
- 数据导出
- 医生远程查看

但数据结构和页面设计需要预留未来扩展空间。

## 2. 用户与场景

### 2.1 目标用户

- 新生儿宝宝的妈妈
- 新生儿宝宝的爸爸
- 参与日常照护的家庭成员

### 2.2 核心使用场景

- 每次喂奶后记录喂养类型、奶量和时间。
- 每次换尿布后记录尿尿、便便、量和颜色。
- 每次测黄疸后记录 3 个黄疸值，并查看平均值和风险标志。
- 每天查看宝宝奶量是否达标。
- 通过趋势图查看最近几天的喂养、排泄和黄疸变化。

## 5. 宝宝档案

### 5.1 功能说明

宝宝档案用于记录基础信息，并作为奶量达标判断和黄疸风险判断的基础。

第一版只支持一个宝宝。

### 5.2 字段

必填字段：

- 宝宝昵称
- 出生日期
- 出生时间
- 当前体重，单位 kg
- 出生孕周

可选字段：

- 头像（第一版本地保存，后续接入云端或服务端上传）
- 性别
- 出生体重，单位 kg
- 是否早产
- 是否存在黄疸高风险因素
- 备注

### 5.3 规则

- 如果缺少出生日期或出生时间，无法计算日龄和出生后小时数。
- 如果缺少当前体重，无法估算每日奶量是否达标。
- 如果缺少出生孕周，无法自动判断黄疸风险。

## 6. 今日概览

### 6.1 功能说明

今日页展示当天最重要的数据，让用户一眼了解宝宝今天的情况。

### 6.2 展示内容

- 今日总奶量
- 今日喂养次数
- 今日奶量状态：偏低 / 达标 / 偏高 / 无法判断
- 今日尿尿次数
- 今日便便次数
- 最近一次黄疸平均值
- 黄疸风险标志：正常 / 低危 / 中危 / 高危 / 无法判断 / 不适用
- 距离上次黄疸记录的时间
- 黄疸较上次是否上升

### 6.3 快捷操作

- 记喂奶
- 记大小便
- 记黄疸

### 6.4 首页提示示例

- 今日奶量 520 ml，达标。
- 今日奶量 360 ml，低于建议范围。
- 最近黄疸平均值 8.6 mg/dL，低危。
- 最近黄疸较上次上升，请持续观察。

## 7. 喂养记录

### 7.1 新增喂养记录

字段：

- 喂养类型：母乳 / 奶粉
- 奶量，单位 ml
- 喂养时间，默认当前时间
- 备注，可选

### 7.2 交互要求

- 奶量支持快捷选择，如 30 / 60 / 90 / 120 ml。
- 奶量支持手动输入。
- 母乳也按 ml 记录，可由用户估算。
- 保存后更新今日概览和趋势数据。

### 7.3 每日统计

按自然日统计：

- 总奶量
- 总喂养次数
- 母乳总量
- 奶粉总量
- 母乳次数
- 奶粉次数
- 奶量状态

### 7.4 奶量达标估算

第一版采用基于体重和日龄/月龄的目标区间，不使用单一固定值。

输入：

- 当前体重 kg
- 出生后日龄 / 月龄
- 当日总奶量 ml

输出：

- 建议奶量下限
- 建议奶量上限
- 今日状态：偏低 / 达标 / 偏高 / 无法判断

计算思路：

```text
每日建议奶量区间 = 当前体重 kg × 日龄/月龄对应 ml/kg/day 区间
```

判断规则：

- 当日总奶量低于建议下限：偏低
- 当日总奶量位于建议区间：达标
- 当日总奶量高于建议上限：偏高
- 缺少体重或出生日期：无法判断

页面文案需要注明：

```text
该结果仅作为家庭记录参考，不替代医生建议。
```

### 7.5 喂养趋势图

- 奶量建议区间参考线

## 9. 黄疸记录

### 9.1 功能优先级

由于当前宝宝处于新生儿阶段，黄疸模块在第一版中属于高优先级功能，需要在首页突出展示最近状态和趋势变化。

### 9.2 新增黄疸记录

字段：

- 黄疸值 1，单位 mg/dL
- 黄疸值 2，单位 mg/dL
- 黄疸值 3，单位 mg/dL
- 平均值，系统自动计算
- 测量方式：经皮 / 血清 / 未知
- 记录时间，默认当前时间
- 自动风险标志
- 备注，可选

### 9.3 平均值计算

```text
平均黄疸值 = (黄疸值 1 + 黄疸值 2 + 黄疸值 3) / 3
```

规则：

- 平均值保留 1 位小数。
- 三个值都必填。
- 输入值只支持 mg/dL。

### 9.4 黄疸自动风险判断

黄疸风险不能只根据固定数值判断，需要结合：

- 出生后小时数
- 出生孕周
- 黄疸平均值
- 是否存在神经毒性风险因素
- 测量方式

第一版风险输出：

- 正常
- 低危
- 中危
- 高危
- 无法判断
- 不适用

判断规则：

- 如果宝宝孕周小于 35 周：显示不适用，建议医生判断。
- 如果缺少出生时间、出生孕周或黄疸值：显示无法判断。
- 如果数据完整：根据新生儿出生后小时数、孕周、风险因素和黄疸值，比较当前黄疸值与对应风险阈值的距离。

分级含义：

- 正常：距离风险阈值较远。
- 低危：接近风险阈值，但仍有安全距离。
- 中危：接近光疗阈值，需要密切观察。
- 高危：达到或非常接近光疗阈值，建议尽快咨询医生。

页面文案：

```text
本结果仅用于家庭记录和观察，不能替代医生判断。
```

高危文案：

```text
当前黄疸值较高，建议尽快咨询医生。
```

## 12. MVP 验收标准

### 12.1 宝宝档案

- 可以创建并编辑一个宝宝档案。
- 可以填写出生日期、出生时间、体重和出生孕周。
- 缺少关键字段时，相关判断显示无法判断。

### 12.2 喂养模块

- 可以新增母乳 / 奶粉记录。
- 可以填写奶量和时间。
- 可以统计当天总奶量和总次数。
- 可以根据宝宝体重和日龄/月龄估算奶量状态。
- 可以查看近 7 / 14 / 30 天喂养趋势。

### 12.3 大小便模块

- 可以记录尿尿、便便或两者同时发生。
- 可以选择尿量 / 粪量的小、中、大。
- 可以选择颜色。
- 可以统计当天尿尿次数和便便次数。
- 可以查看大小便趋势图。

### 12.4 黄疸模块

- 可以输入 3 个 mg/dL 黄疸值。
- 系统自动计算平均值。
- 系统根据宝宝信息和黄疸值自动生成风险标志。
- 可以查看最近一次黄疸状态。
- 可以查看黄疸趋势图。

### 12.5 通用能力

- 所有记录可查看、编辑、删除。
- 所有记录可按日期筛选。
- 首页能显示今日核心概览。
- 第一版不需要登录、不需要同步、不需要提醒。

## 14. 医疗提示原则

小程序中的奶量达标、颜色提示和黄疸风险标志都只作为家庭记录和观察参考。

所有涉及健康风险的页面都需要避免使用确诊、必须治疗、不用担心等绝对化表述。

推荐统一提示：

```text
本结果仅用于家庭记录和观察，不能替代医生判断。如宝宝状态异常，请及时咨询医生。
```
