# 宝宝喂养记录小程序 UI 设计说明

## 设计方向

- 风格：极简、低噪、纸张感，适合新生儿家庭在夜间或碎片时间快速记录。
- 核心记忆点：今日页用一个超大奶量数字承载主状态，其他信息退到次级层级。
- 情绪：温和但不幼稚，避免过度可爱化，健康风险提示保持克制。

## 页面稿

- `01 今日`：今日总奶量、达标状态、尿便次数、最近黄疸和三类快捷记录。
- `02 记录`：三类记录分段切换，当前展示喂养历史，倒序卡片便于编辑和删除。
- `03 记喂奶`：底部 Sheet 表单，支持母乳/奶粉切换、快捷奶量、手动输入、时间和备注。
- `04 趋势`：近 7/14/30 天切换，展示奶量建议区间、喂养次数和黄疸风险区间。
- `05 宝宝`：单宝宝档案，突出日龄、体重、孕周和黄疸风险因素。

## 视觉规范

- 主色：`#20211F` 墨黑，用于文字、边框、主按钮。
- 背景：`#F7F7F2` 低饱和纸色，页面主体使用 `#FFFEF9`。
- 状态色：绿色表示达标，黄色表示观察，珊瑚色表示高关注，蓝色表示趋势辅助信息。
- 字体：标题使用宋体气质字形，正文使用系统中可用的 Avenir/PingFang 组合。
- 圆角：卡片与按钮控制在 8px 左右，底部导航和手机外框允许更大圆角。

## 开发拆分建议

- `risk-badge`：统一处理正常、低危、中危、高危、无法判断、不适用。
- `overview-metric`：今日页大数字和建议区间。
- `record-sheet`：喂养、尿便、黄疸新增表单都复用底部 Sheet 结构。
- `trend-chart`：第一版可先用 canvas 或小程序 chart 组件实现折线、散点和风险背景。
- 所有健康文案统一使用 PRD 的提示原则，不输出绝对化医疗结论。
